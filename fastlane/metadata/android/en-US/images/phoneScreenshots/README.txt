Drop phone screenshots here for the F-Droid / store listing.

Requirements:
- File names: 1.png, 2.png, 3.png ... (order shown in the listing)
- Format: PNG or JPG
- Portrait phone screenshots of the app UI (not DevTools).

Suggested shots:
  1.png  Main screen (URL + QR code, server running)
  2.png  Web file manager in a browser (file list)
  3.png  In-browser preview (image or video)
  4.png  Gallery view / thumbnails
  5.png  Trash or Share dialog

These are picked up automatically by the F-Droid build (fastlane metadata
layout) and can be referenced from README.md.
